## Setup and Installation

### 1. Python Environment
It is recommended to use a virtual environment to manage dependencies:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: .\venv\Scripts\activate
```

### 2. Dependencies
Install the required packages using pip:
```bash
pip install -r requirements.txt
```
*(Note: If requirements.txt is not present, check individual subfolders for specific needs.)*

### 3. Execution
Look for Jupyter notebooks (.ipynb) or Python scripts (.py) within the project folders to start the experiments.
